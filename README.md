# Harsh Rathod Portfolio

A personal portfolio website built with React and React-Bootstrap.

![Portfolio preview](./Images/readme-img1.png)

## Overview

This project showcases personal information, technical skills, projects, resume, and coding profiles in a responsive multi-section layout.

## Features

- Multi-section portfolio layout
- Responsive design for desktop and mobile
- GitHub activity calendar
- LeetCode stats and streak card
- Resume preview and download
- Smooth navigation between sections

## Tech Stack

- React
- React-Bootstrap
- React Router
- React Icons
- React GitHub Calendar
- React PDF
- Typewriter Effect
- Particle animation library

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

```bash
npm install
```

### Run the app

```bash
npm start
```

The app runs in development mode at [http://localhost:3000](http://localhost:3000).

### Build for production

```bash
npm run build
```

## Project Structure

- `src/components` contains the UI sections and shared components
- `src/Assets` contains images, icons, and resume files
- `public` contains the static app shell

## Customization

Update the following files to personalize the portfolio:

- `src/components/Home/Home.js`
- `src/components/About/About.js`
- `src/components/About/Github.js`
- `src/components/About/Leetcode.js`
- `src/components/Resume/ResumeNew.js`
